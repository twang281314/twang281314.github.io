MongoDB备份和还原

发表于 2015-09-27   |   分类于 MongoDB

MongoDB数据备份

在Mongodb中使用mongodump命令来备份数据，该命令可以可以导出所有数据到制定目录

命令语法

mongodump -h dbhost -d dbname -o dbdirectory

-h MongDB所在服务器地址，例如：127.0.0.1，可以指定端口号：127.0.0.1:27017

-d 需要备份的数据库实例，例如：mytest

-o 备份的数据存放位置，例如：c:\mongodb_data\dump，当然该目录需要提前建立，在备份完成后，系统会自动在dump目录下建立一个mytest目录，这个目录里面存放该数据库实例的备份数据

MongoDB数据恢复

使用mongorestore命令来恢复数据

命令语法

mongorestore -h dbhost -d dbname –directoryperdb dbdirectory

-h MongoDB服务器地址
-d 需要恢复的数据库实例名称 可以和备份时候的不一致
–directoryperdb 备份数据所在的位置
